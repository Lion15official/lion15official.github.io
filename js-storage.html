Data Persistence

STEP 01

The setItem Method

localStorage.setItem('theme', 'dark');

Pro Tip

LocalStorage keeps data even after the browser is closed. It's perfect for saving user preferences like Dark Mode or Language!

STEP 02

The getItem Method

const currentTheme = localStorage.getItem('theme');

Important

Always remember that LocalStorage only stores strings. If you want to store an object, you must use JSON.stringify() first.
