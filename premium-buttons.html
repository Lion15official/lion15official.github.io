Premium Glow Buttons

Learn the secret behind the "Lion15" interaction design: shadows, transitions, and movement.

Live Showcase

The Final Effect

Step 1

Static Glow

We use box-shadow to create a soft light around the button. The key is using rgba for transparency.

box-shadow: 0 0 20px rgba(0, 204, 255, 0.4);
Step 2

Hover & Motion

By adding transition and translateY, the button feels alive when the user interacts with it.

transition: 0.3s ease;
transform: translateY(-5px);
Master Class

The Complete CSS Code

Copy this to your style.css to use it anywhere.

/* The Base Button Style */
.btn-premium {
  background: #ffffff;
  color: #000000;
  padding: 12px 25px;
  border-radius: 25px;
  font-weight: 700;
  transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

/* The Glow Interaction */
.btn-premium:hover {
  transform: translateY(-3px);
  box-shadow: 0px 10px 30px rgba(0, 204, 255, 0.5);
  border: 1px solid #00CCFF;
}
