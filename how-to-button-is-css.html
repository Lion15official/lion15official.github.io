How to Style a Modern Button

1. The HTML Structure

First, create a simple anchor or button tag with a class.

<a href="#" class="btn-modern">Explore More</a>

2. The CSS Magic

Copy this to your stylesheet to get the glow and glass effect.

.btn-modern {
  background: rgba(255, 255, 255, 0.05);
  backdrop-filter: blur(10px);
  border: 1px solid rgba(255, 255, 255, 0.1);
  color: white;
  padding: 15px 35px;
  border-radius: 50px;
  transition: 0.4s;
}

.btn-modern:hover {
  background: #00c8ff;
  box-shadow: 0 0 20px rgba(0, 200, 255, 0.5);
}
