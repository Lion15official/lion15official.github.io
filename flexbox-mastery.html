Mastering Flexbox

The ultimate CSS layout tool. Control alignment, spacing, and order with just a few lines of code.

Concept 1

Justify Content (Horizontal)

This property aligns items along the main axis (usually left-to-right).

1
2
3
justify-content: center;
Concept 2

Align Items (Vertical)

This property aligns items along the cross axis (top-to-bottom).

1
2
3
align-items: center;
Pro Tip

The Ultimate Centering Trick

To center any element perfectly in the middle of a container, use this magic combo:

display: flex;
justify-content: center;
align-items: center;
